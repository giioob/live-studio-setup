1设计总原则（先记住这几条）

所有选择背后的逻辑，先对齐心法再看明细。

原则怎么做 / 为什么
质感三要素：光 ≈ 声 > 相机高级感主要来自布光 + 干净声音 + 实景背景，不是相机贵。所以钱优先砸灯光、声学、置景。
相机先省，用 Pocket 3 起步已有 Pocket 3，先零成本开拍；省下 ~1.1 万。等出内容/有回流再买全画幅（兼顾照片+外出）。
分两步走，别一次到位第一阶段把单机位口播+录屏跑通；访谈/多机位/ATEM 作为第二阶段，赚了再补。
实景虚化，不做绿幕绿幕抠像门槛高、易穿帮、且丢掉真实虚化，与影视飓风质感相反。换背景靠换前景软装 + 背景大屏。
预算纪律：剩一半攥手里第一阶段只花 ~1.66 万，剩 ~1.34 万留作跑道 buffer + 未来相机基金。设备能等你赚钱再补。
一室多用同一面背景 + 同一组机位，换前景即在「口播 / 访谈」间切换；南墙做不入镜的运营办公带。

5灯光方案（四灯布光 · amaran）

灯位作用器材

10落地节奏（建议顺序）

  • 第 1 步：北墙窗帘 + 地毯 + 西/南墙吸音 + 防潮三件套（先把环境做对）
  • 第 2 步：东墙陈设（书架/灯带/装饰画/绿植/落地灯）——唯一的门面，慢慢调到满意
  • 第 3 步：灯光四件套到位 + Pocket 3 + DJI Mic 2 + 提词器 → 开拍验证出镜状态
  • 第 4 步：用现有设备试录制式多机位 + 搭一套后期动效模板
  • 第 5 步：出了内容/有回流 → 再决策全画幅相机 + 访谈升级（第二阶段）
先把"能出片"跑通，别等"完美的棚"。卖了再准备 > 等准备好再卖。
